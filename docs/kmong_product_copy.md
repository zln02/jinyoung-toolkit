## 한 마디 소개 (핵심 훅)

**리뷰가 쌓여만 가고 분석할 시간이 없나요?**
한국어 리뷰 1,000개를 단 **30분 만에 자동 분석**합니다.
워드클라우드·감성분석·키워드 추출은 기본이고, **경쟁사 최대 3개까지 비교 테이블 + 액션 아이템**까지 PDF로 납품해드립니다.
세컨드 사이트도, 앱스토어도 전부 지원합니다.

## 가격 정책

### 💰 기본 티어: **10만 원**

**포함 내용:**
- ✓ 단일 제품 리뷰 크롤링 (최대 1,000개)
- ✓ 감성분석 (긍정/부정/중립)
- ✓ 워드클라우드 생성
- ✓ 키워드 추출 (상위 30개)
- ✓ 요약 통계 (평균 평점, 트렌드)
- ✓ PDF 리포트 (A4, 3~5페이지)
- ✓ CSV 원본 데이터
- ✓ ZIP 패키지 (전체)

**예시:** 아마존 신발 상품 리뷰 500개 → 감성분석 PDF + CSV 납품

---

### 🏆 프리미엄 티어: **30만 원**

**포함 내용:**
- ✓ 기본 티어의 모든 것 +
- ✓ **경쟁사 비교 리포트** (우리 1개 + 경쟁 최대 3개, 총 4개 URL)
- ✓ 경쟁사 분석 결과 병렬 비교
- ✓ "우리가 이기는 포인트" 자동 도출
- ✓ "개선 필요 포인트" 액션 아이템
- ✓ 비교 테이블 (감성도, 주요 키워드, 트렌드)
- ✓ 전략적 인사이트 (경쟁 분석서)
- ✓ 확장 PDF 리포트 (A4, 7~10페이지)
- ✓ 모든 제품의 원본 CSV (4개)
- ✓ ZIP 패키지 (전체)

**제약:** 같은 플랫폼 내에서만 비교 가능
예: 쿠팡 vs 쿠팑 (쿠팡 vs 11번가는 불가)

**예시:** 11번가 로보락 vs LG/삼성/디베아 → 비교 분석 PDF + 4개 CSV 납품
샘플: `samples/demo_comparison_roborock/comparison_report.pdf` (3페이지, 0건 제품 자동 제외, 권장 개선 포인트 4개)

## 진행 절차

```
1️⃣ 주문 접수
   → 크몽 메시지 또는 카톡으로 주문 확인

2️⃣ URL 및 옵션 전달 (24시간 이내)
   → 판매자에게 상품 URL, 티어(기본/프리미엄), 선호 분석 항목 전달
   → 기본 티어: 1개 URL
   → 프리미엄 티어: 4개 URL (우리 1 + 경쟁 3)

3️⃣ 크롤링 + 분석 (1~2일 소요)
   → 자동화 프로세스:
      [크롤링] → [감성분석] → [키워드 추출] → [PDF 생성] → [패키지화]
   → 대기 시간: 최대 60초 (사이트 렌더링)
   → 실패 시: 재시도 또는 환불 협의

4️⃣ 납품 (최대 3영업일)
   → 카톡/이메일로 ZIP 파일 전달
   → 다운로드 링크 또는 직접 송금

5️⃣ 무료 수정 (납품 후 3일 이내, 1회)
   → 차트 색상 변경, 리포트 항목 추가 등 간단한 수정
   → 전체 재분석은 추가 비용 발생 가능
```

## 왜 택했을까? (판매 근거)

### ✨ 자동화의 힘
- **수동 작업 → 자동 크롤링**: 프리랜서가 하나하나 복사-붙여넣기할 필요 없음
- **하이브리드 감성분석**: 기본은 별점 기반 룰(1~2점 = 부정, 4~5점 = 긍정) + 키워드 사전. 고객이 원할 경우 Anthropic Haiku LLM 추가(1건당 약 1원). 별점이 텍스트("만족도 : 100%")라도 프리셋의 `rating_parse_pattern` 으로 자동 정규화.
- **30분 처리**: 리뷰 1,000개 분석을 수동으로 하면 2주가 걸림

### 🎯 경쟁사 비교 (차별화)
- 타 대행업체는 "단일 제품 분석"만 제공
- **우리는 경쟁 구도를 한눈에**: "시장에서 우리의 포지션은?"
- B2B 스타트업·소호 사업자가 마케팅 의사결정에 직접 활용

### 📊 프리미엄 인사이트
- 단순 워드클라우드가 아닌 **액션 아이템**: "리뷰에서 본 약점 3가지 개선하면 경쟁사 이길 수 있음"
- PDF 품질: 차트 + 색상 표준화로 전문성 극대화

### ⚡ 빠른 턴어라운드
- 자동화 덕분에 **1~2일 납품** (타 대행은 3~7일)
- 숨고/크몽 초기 평점 3개 → "빠른 업체"라고 입소문 → 반복 주문

### 💰 수익성
- Haiku LLM 비용: 1,000개 리뷰당 약 500원
- 크롤링/서버 비용: 무시할 수 있는 수준
- **마진율 99%** → 가격 경쟁에서 이길 수 있음

## 자주 묻는 질문 (FAQ 참조)

- Q: 비개발자도 주문 가능한가요?
  - A: 네, URL만 전달하면 판매자가 전부 처리합니다.

- Q: 리뷰가 50개만 있어도 분석 가능한가요?
  - A: 최소 20개 권장. 그 이하면 통계 유의성 부족으로 협의 필요.

- Q: 크롤링이 실패하면 어떻게 되나요?
  - A: 전액 환불 또는 다른 사이트로 대체 가능.

- Q: 데이터는 얼마나 보관되나요?
  - A: 납품 후 7일 뒤 자동 삭제 (개인정보 보호법 준수).
