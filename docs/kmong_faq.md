# jinyoung-toolkit 크몽 FAQ

---

## 📋 상품 개요 관련

### Q1: 이 서비스는 정확히 뭔가요?

**A:** 한국어 리뷰를 자동으로 수집·분석·시각화하는 AI 서비스입니다.

- **수집**: 아마존, 쿠팡, 네이버 등 14개 사이트에서 자동 크롤링
- **분석**: 감성분석(긍정/부정/중립), 키워드 추출, 워드클라우드 생성
- **경쟁사 비교**: (프리미엄) 우리 상품 vs 경쟁사 3개 병렬 비교
- **납품**: PDF 리포트 + 원본 CSV 데이터

약 30분 내에 리뷰 1,000개를 처리합니다.

---

### Q2: 비개발자도 주문할 수 있나요?

**A:** 네, 전혀 개발 지식이 없어도 됩니다.

판매자가 모든 작업을 대행합니다.
1. 상품 URL 알려주기 (이메일/카톡)
2. 1~2일 기다리기
3. 납품받은 PDF + CSV 사용하기

그 이상의 작업은 없습니다.

---

### Q3: 기존 프리랜서 대행과 뭐가 다른가요?

**A:** 크게 3가지입니다.

1. **경쟁사 비교 기능**: 우리 vs 경쟁 3개를 한 PDF에서 비교 (차별화)
2. **처리 속도**: 타 업체는 3~7일, 우리는 1~2일 (자동화 덕분)
3. **가격**: 기본 10만원 vs 타 업체 15~30만원 (비용 효율)

특히 **경쟁사 비교**는 매우 드문 서비스입니다.

## 💰 가격 관련

### Q4: 왜 10만원과 30만원 두 가지인가요?

**A:** 두 가지 필요성이 있기 때문입니다.

| 경우 | 추천 티어 | 이유 |
|------|---------|------|
| 우리 상품만 분석하고 싶음 | 기본 (10만원) | 경쟁사 비교 불필요 |
| 경쟁사 vs 우리를 비교하고 싶음 | 프리미엄 (30만원) | 마케팅 의사결정에 필요 |

**기본**: 자사 상품 리뷰 1,000개 분석 + PDF 리포트
**프리미엄**: 기본 + 경쟁사 3개 병렬 비교 + 액션 아이템

| 항목 | 비용 |
|------|------|

### Q6: 환불 정책은 어떻게 되나요?

**A:** 다음과 같습니다.

| 상황 | 환불 |
|------|------|
| 크롤링 실패 (사이트 접근 불가) | ✓ 100% 환불 |
| 리뷰 20개 미만 (통계 유의성 부족) | ✓ 100% 환불 또는 협의 |
| 작업 완료 후 단순 변심 | ✗ 환불 불가 |
| 분석 결과 불만족 | △ 무료 수정 1회 제공 |

**크롤링 시작 전**: 전액 환불 가능
**크롤링 시작 후**: 분석이 진행되면 환불 불가

### Q8: 네이버쇼핑/쿠팡이 "제한적"이라는 건 뭔가요?

**A:** 사이트마다 봇 차단 정책이 다르기 때문입니다.

**제약:**
- iframe 또는 자바스크립트 기반 리뷰 로딩 → 일부 페이지만 수집 가능
- 동적 로딩으로 인한 속도 저하 (최대 60초 대기)
- 일시적 IP 차단 위험

**우리가 어떻게 대응하나요?**
- 0건 수집된 제품은 PDF가 자동으로 표·차트에서 제외하고 상단에 "크롤링 실패 제품" 경고로 알려드려요. 부분 결과만 받아도 즉시 의사결정에 쓸 수 있어요.
- 사전 검증 단계에서 수집 가능 여부를 확인 → 불가능하면 시작 전에 환불하거나 다른 사이트로 협의합니다.

**해결 방법:**
1. 다른 사이트 URL 사용 (11번가, YES24, 멜론, CGV 등 — 비교적 안정)
2. CSV로 직접 리뷰 데이터 제공 (가장 확실)
3. 재시도 (IP 차단 해제 후)

## 📊 분석 관련

### Q9: 리뷰가 너무 적으면 분석이 안 되나요?

**A:** 네, 일정 수준 이상이 필요합니다.

| 리뷰 개수 | 분석 가능 | 비고 |
|----------|---------|------|
| 200개 이상 | ✓ 강력 추천 | 통계적으로 충분 |
| 50~200개 | △ 가능하지만 주의 | 추세는 보이나 신뢰도 낮을 수 있음 |
| 20~50개 | △ 고민 | 환불 또는 추가 협의 권장 |
| 20개 미만 | ✗ 불가 | 통계적 의미 없음 → 전액 환불 |

**권장:** 최소 50개 이상 (가능하면 200개+)

---

### Q10: 분석 결과 정확도는 어떻게 되나요?

**A:** 두 가지 분석 방식을 함께 사용해 정확도를 끌어올립니다.

**분석 방식:**
1. **별점 기반 룰** (기본, 무료) — 1~2점 부정, 3점 중립, 4~5점 긍정. 별점이 텍스트 형태("만족도 : 100%", "★★★★★")여도 프리셋의 `rating_parse_pattern` 으로 자동 정규화합니다.
2. **키워드 사전 보강** — 한국어 긍정/부정 키워드 사전(내부 구축)으로 키워드별 가중치 추가
3. **(옵션) Claude Haiku LLM** — 1건당 약 1원, 반어·풍자 포함 어려운 케이스에 추천

**한계 (솔직한 안내):**
- 반어, 풍자, 비꼬는 표현 → 별점이 낮으면 부정으로 분류되지만 의미는 다를 수 있음
- 외국어 혼용, 특수문자 많은 리뷰 → 키워드 추출 정확도 저하
- 별점이 없는 사이트(블로그, 카페) → LLM 옵션 권장

**참고:** 100% 정확도를 보장하지 않으며, 분석은 "경향" 및 "차이점" 파악용입니다. 의사결정에 직접 사용하기 전에 PDF의 키워드 Top 10을 함께 검토하세요.

## 📥 납품 및 처리 관련

### Q12: 분석 결과는 어떻게 받나요?

**A:** 3가지 형태로 납품됩니다.

| 형식 | 내용 | 용도 |
|------|------|------|
| **PDF 리포트** | A4 컬러, 7~10페이지 | 이해관계자 공유, 인쇄 가능 |
| **CSV 데이터** | 원본 리뷰 + 감성 레이블 | 추가 분석, 가공 |
| **ZIP 패키지** | 위의 모든 파일 | 한 번에 다운로드 |

**전달 방법:**
1. 크몽 메시지로 다운로드 링크 제공 (Google Drive / OneDrive)
2. 또는 카톡 파일 전송
3. 또는 이메일 송금 (사전 협의)

**다운로드 기한:** 30일 (그 후 자동 삭제)

**A:** 보통 1~2일, 최대 3일입니다.

| 단계 | 소요 시간 | 비고 |
|------|---------|------|
| 주문 접수 | 즉시 | 크몽 알림 |
| URL 수신 & 검증 | 2~4시간 | 크롤링 가능 여부 사전 확인 |
| 크롤링 | 30분~2시간 | 리뷰 개수 & 사이트에 따라 |
| 감성분석 & 키워드 추출 | 10~30분 | LLM 처리 시간 |
| PDF 생성 | 5분 | 자동화 |
| 납품 | 즉시 | 압축 & 링크 생성 |
| **총 소요**: | **1~2일** | 대기 포함 최대 3일 |

**단, 다음의 경우 연장될 수 있습니다:**
- 리뷰 3,000개 이상 (추가 1일 소요)
- 사이트 점검 또는 IP 차단 (재시도 필요)
- 주말/공휴일 (다음 영업일 처리)

- ✗ PowerPoint 별도 생성

### Q16: 리뷰 데이터 저작권은 어떻게 되나요?

**A:** 수집된 리뷰는 원본 저작권자에게 있습니다.

**우리의 역할:**
- ✓ 수집 자동화 & 분석 도구 제공
- ✓ 분석 결과(리포트)는 고객 소유
- ✗ 리뷰 자체의 재판매 금지
- ✗ 대량 재배포 금지

**고객의 용도:**
- ✓ 자사 마케팅 자료 (내부 공유)
- ✓ 경영진 보고서
- ✗ 타사 판매 또는 공개 배포

## 🔄 경쟁사 비교 (프리미엄) 관련

### Q17: 경쟁사 비교는 몇 개까지 가능한가요?

**A:** 프리미엄 티어에서 우리 1개 + 경쟁 최대 3개 (총 4개)입니다.

**제약:**
- 같은 플랫폼 내에서만 비교 가능
- 예: ✓ 쿠팡 vs 쿠팡 (3개 경쟁사)
- 예: ✗ 쿠팡 vs 네이버쇼핑 (다른 사이트)

**이유:** 사이트별로 리뷰 스타일, 평점 분포가 다르기 때문에, 같은 사이트 내에서만 정확한 비교가 가능합니다.

---

### Q18: 경쟁사 3개를 모두 포함해야 하나요? 1개만 비교 가능한가요?

**A:** 아니요, 자유입니다.

| 구성 | 가능 여부 | 가격 |
|------|---------|------|
| 우리만 분석 | ✓ | 10만원 (기본) |
| 우리 vs 경쟁 1개 | ✓ | 30만원 (프리미염) |
| 우리 vs 경쟁 2개 | ✓ | 30만원 |
| 우리 vs 경쟁 3개 | ✓ | 30만원 |

프리미엄은 **최대 3개까지**이지, 꼭 3개를 해야 하는 건 아닙니다.

---

### Q19: 경쟁사 비교 리포트에는 뭐가 들어가나요?

**A:** 다음과 같습니다.

**포함 내용:**
1. **병렬 분석 테이블**
   - 우리 vs 경쟁 3개의 워드클라우드 4개 (한눈에 비교)
   - 감성도 비교 (긍정률, 부정률)
   - 주요 키워드 비교

2. **"우리가 이기는 포인트"**
   - 우리만 자주 언급되는 강점 (예: "빠른 배송", "좋은 품질")
   - 경쟁사에는 없는 차별화 요소

3. **"우리가 지는 포인트"**
   - 경쟁사는 자주 나오지만 우리는 적게 나오는 약점
   - 개선 필요 항목

4. **액션 아이템**
   - "다음 분기 개선할 3가지"
   - 우선순위 및 예상 효과

5. **트렌드 분석**
   - 시간대별 리뷰 발생 추이 (우리 vs 경쟁)
   - 계절성 패턴

**전체 7~10페이지 PDF로 제공**

## 🆘 문제 해결

### Q20: 크롤링이 실패했다고 나왔어요. 뭐가 문제인가요?

**A:** 여러 원인이 있습니다.

| 원인 | 해결 방법 |
|------|---------|
| IP 차단 | 시간 경과 후 재시도 (보통 1시간) |
| 사이트 점검 | 사이트 복구까지 대기 후 재시도 |
| URL 오류 | 올바른 상품 URL 재확인 후 재시도 |
| 리뷰 없음 | 다른 상품 또는 다른 사이트로 변경 |
| 사이트 구조 변경 | 업체 공지 (프리셋 업데이트 예정) |

**대응:**
1. 1차 재시도 (비용 없음)
2. 2차 이상: 환불 또는 다른 상품으로 대체
