| **총 원가** | — | **약 120원** |

### 수익 계산

| 항목 | 기본 (10만원) | 프리미엄 (30만원) |
|------|--------|---------|
| 판매가 | 100,000원 | 300,000원 |
| 크몽 수수료 (20%) | -20,000원 | -60,000원 |
| 원가 (4개 URL) | -120원 × 4 = -480원 | -480원 |
| **순 마진** | **~80,000원** | **~240,000원** |
| **마진율** | **80%** | **80%** |

### 시간 가치

| 항목 | 소요 시간 |
|------|---------|
| 크롤링 + 분석 자동화 | 30분~2시간 (자동) |
| 사용자 상담 & 검증 | 30분~1시간 |
| 납품 & 지원 | 10분 |
| **총 직접 시간** | **1~3시간** |

**시간당 수익:**
- 기본: 80,000원 ÷ 2시간 = **40,000원/시간**
- 프리미엄: 240,000원 ÷ 2시간 = **120,000원/시간**

이는 한국 프리랜서 평균 시급(15,000~30,000원)보다 높으므로 경쟁력 있는 수준입니다.

## 4️⃣ 리스크 공지 & 운영 유의사항

### 상품 페이지에 명시할 리스크

다음 텍스트를 상품 설명 맨 아래 **주의** 섹션에 추가:

```
⚠️ 주의사항

1. 일부 사이트(네이버쇼핑, 쿠팡)는 반봇 기술이 있어 IP 차단 발생 가능
   → 이 경우 1~2시간 대기 후 재시도하거나, 다른 사이트로 변경 권장

2. 사이트가 JavaScript 렌더링을 사용하면 최대 60초 대기 시간 발생 가능
   → 정상 작동이나 처리 속도 느린 경우로 분류

3. 사이트 웹 구조 변경으로 크롤링 실패 시 → 전액 환불 정책 적용
   → 이 경우 업체에서 프리셋 업데이트 후 재시도 가능

4. 데이터는 납품 후 7일 뒤 자동 삭제됨 (개인정보보호법 준수)

5. 분석 결과는 경향 파악용이며, 100% 정확도를 보장하지 않음
   (감성분석 정확도 약 95~98%)
```

## 7️⃣ 문제 해결 플로우차트

### 크롤링 실패 시

```
크롤링 실패
  ├─ IP 차단? → 1시간 대기 후 재시도
  ├─ 사이트 점검 중? → 복구까지 대기
  ├─ URL 오류? → 사용자에게 올바른 URL 재요청
  ├─ 리뷰 없음? → 다른 상품 또는 사이트로 변경 제안
  └─ 사이트 구조 변경? → 환불 (프리셋 업데이트 예정)
```

### 고객 불만족 시

```
분석 결과 불만족
  ├─ 차트 보기 어려움? → 무료 수정 (3일 내 1회)
  ├─ 데이터 오류? → 원본 CSV 확인 후 재분석
  ├─ 경쟁사 비교 잘못됨? → 프리미엄 유료 재작업 (+10만원) 또는 환불
  └─ 기타 만족도 낮음? → 추가 1회 무료 수정 협의
```
